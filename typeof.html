JS中有哪些识别类型的方法？

1:typeof 识别除了Null以外的标准类型； 不能识别Function以外的具体对象类型；

2:instanceof 识别内置对象类型； 不能识别原始类型； 可以识别自定义对象类型；

3:Object.prototype.toString.call() 可以识别标准类型以及内置对象类型； 不能识别自定义对象类型；

4:constructor 识别除了Undefined/Null之外的标准类型； 识别内置对象类型； 识别自定义对象类型；

    //constructor 配合代码
	function getConstructorName(obj){
    	return (obj===null||obj===undefined)?obj:(obj.constructor&&obj.constructor.toString().match(/function\s*([^)]*)/)[1]);    
	}
